SPRAYWALL IMAGE
================

To use your own spraywall photo:

1. Take a clear, front-facing photo of your spraywall
2. Save it in this folder as one of:
   - spraywall.jpg
   - spraywall.png
   - spraywall.svg (current placeholder)

3. Update the image path in /src/main.ts (line ~124) to match your filename:
   src="/spraywall.jpg"   (or .png, .svg, etc.)

Recommended image specs:
- Format: JPG or PNG
- Resolution: 2000-4000px wide
- Well-lit, front-facing shot
- File size: Under 5MB

The placeholder spraywall.svg file can be deleted once you add your own image.
